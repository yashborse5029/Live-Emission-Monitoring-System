# Live-Emission-Monitoring-System
IoT-based vehicle emission monitoring system using ESP32 and gas sensors for real-time pollution analysis and environmental monitoring.

# 🌍 Live Emission Monitoring System (LEMS)

### ESP32-Based IoT System for Real-Time Vehicle Emission Monitoring

---

## 📖 Project Overview

The **Live Emission Monitoring System (LEMS)** is an IoT-based embedded solution designed to continuously monitor vehicle exhaust emissions in real time. The system uses an ESP32 microcontroller along with gas and particulate matter sensors to detect harmful pollutants generated by vehicles.

Sensor data is processed by the ESP32 and can be displayed locally or transmitted over Wi-Fi for remote monitoring. The system enables early detection of excessive emissions, helping reduce environmental pollution and supporting preventive vehicle maintenance.

The project demonstrates the integration of embedded systems, IoT communication, environmental sensing, and real-time data acquisition to build a practical smart transportation solution.

---

## 🎯 Project Objectives

- Monitor vehicle emissions in real time
- Detect harmful gas concentrations
- Measure particulate matter levels
- Enable IoT-based remote monitoring
- Improve environmental awareness
- Support preventive vehicle maintenance
- Develop a low-cost embedded monitoring solution
